Roadmap

We've got a couple things planned for the next couple months.

High Priority

  • Creating a better speech-to-text model

Medium Priority

  • For the others like me who'd like to see how Scripty works: the raw result of transcriptions in JSON format (requires attach files)
  • More premium features + automated premium subscriptions (requires view linked accounts + view username & avatar)
  • If the user is opted in (opt out default), storing their messages and feeding it back into a model
  • As above, if the user is opted in, storing their audio and feeding it back into a model
  • Crowd-sourcing an audio dataset by allowing people to listen to stored audio to verify its transcription

Low Priority

  • Text-to-speech on demand
