Scripty Premium

Scripty offers basic premium plans that don't do much right now, but that will later come with lots of features.

Why are they so expensive???

Speech-to-text uses a truly insane amount of processing power, and our single server has limited resources. Despite being relatively powerful (AMD Ryzen 9 3900 + 128GB RAM), it's estimated we'll only be able to do about 100 concurrent transcriptions with it. We can work to improve the STT algorithm and our computing capacity with this money.

Details

You will be redirected to GitHub to subscribe to premium via GitHub Sponsors. To claim this, for now you must contact the core devs in the support server after subscribing. In the future this process will be streamlined that way you just need to log in with Discord and a linked GitHub account.
